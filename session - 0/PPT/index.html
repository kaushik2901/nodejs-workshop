What is Node.js?

Node.js is an open-source, cross-platform, back-end JavaScript runtime environment that runs on the V8 engine and executes JavaScript code outside a web browser.

- Wikipedia

Let's talk on Node.js Architecture!

  • Before node javascript code was only running inside browser
  • Browser architecture: Runtime environment, Javascript engine
  • V8 in Chrome, Chakra in Edge, SpiderMonkey in Firefox
  • document and window object to interact with lower level APIs
  • Ryan Dahl in 2009
  • took v8 (fastest engine) and put that into c++ program
  • file system module, http server module etc.

Advantages

  • Super-fast and highly scalable application
  • The accessibility of a single programming language
  • Cleaner and more consistent code base
  • Great for fast to market development (ideal for startups, proof of concepts, prototypes, etc.)
  • Large ecosystem of open source libraries

Limitations

  • Slow to process synchronous CPU intensive task
  • Lots of node developers out there!
  • Hard to understand asynchronous nature of APIs for beginners
  • Large ecosystem of open source libraries
